## How to Use This File

### Adding a New Expense

1. **Quick Entry**: Add to the appropriate week table
2. **Detailed Entry**: Create a new file using [financial-template.md](../../Resources/Templates/financial-template.md)
3. **Update Totals**: Recalculate week/category totals monthly

### Monthly Workflow

1. Log expenses daily or weekly
2. Categorize all transactions
3. Update category breakdown at month-end
4. Compare with budget.md
5. Identify patterns and adjustments needed

### Tips

- Log expenses immediately (or keep receipts to log daily)
- Use consistent category names
- Tag tax-deductible and reimbursable expenses
- Review weekly to catch any missed transactions
- Export to CSV for advanced analysis if needed
