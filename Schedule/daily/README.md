# Daily Logs

## Purpose

Daily logs are your daily planning and reflection tool. Each day gets its own file with structure for planning, tracking, and reviewing.

## Auto-Generation

Once Google Calendar integration is configured, daily log files will be automatically generated at midnight with:
- Calendar events for the day
- Tasks due that day
- Standard daily log structure

## File Naming

Files are named by date: `YYYY-MM-DD.md`

Example: `2026-03-11.md`

## Daily Log Structure

Each daily log includes:

### Planning (Morning)
- Today's Focus: Top 3 priorities
- Calendar Events: Synced from Google Calendar
- Tasks: Organized by priority
- Time Blocking: Hour-by-hour plan

### Tracking (During Day)
- Completed tasks (check them off)
- Quick notes and captures
- Meeting notes links

### Review (Evening at 8:00 PM)
- Wins & Accomplishments
- Challenges & Blockers
- Learning & Insights
- Tomorrow's Priorities

## Daily Workflow

### Morning (15 minutes)
1. Open today's daily log
2. Review calendar events
3. Set top 3 priorities
4. Plan time blocks
5. Prepare materials needed

### During Day
- Check off completed tasks
- Add quick captures
- Update as needed
- Stay on track with time blocks

### Evening Review (10 minutes)
1. Complete review sections
2. Mark daily review as done
3. Rate the day
4. Plan tomorrow's top 3 priorities
5. Process any quick captures to Inbox

## Tips for Success

1. **Review first thing** - Start day with clarity
2. **Three priorities max** - Less is more
3. **Time block realistically** - Account for interruptions
4. **Review before bed** - Closure and planning
5. **Keep it simple** - Don't overthink

## Using the Template

Create today's log:
```bash
cp ../../Resources/Templates/daily-log-template.md 2026-03-11.md
```

Or wait for auto-generation once calendar sync is active.
