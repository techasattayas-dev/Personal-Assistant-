# Monthly Reviews

## Purpose

Monthly reviews are your strategic check-in time. This is when you:
- Review financial health (budget vs actual)
- Assess progress on Area goals
- Clean up and organize Resources
- Review and update long-term goals
- Perform system maintenance

## Schedule

**Day**: 1st of each month
**Time**: 10:00 AM
**Duration**: 1-2 hours

## File Naming

Files are named by year and month: `YYYY-MM.md`

Example: `2026-03.md` (March 2026)

## Monthly Review Structure

### 1. Financial Review (30 minutes)
- Compare budget vs actual spending
- Review all expense categories
- Check savings and investment progress
- Update financial goals
- Plan next month's budget

### 2. Area Assessment (30 minutes)
- **Finance**: Goals progress, budget adjustments
- **Health**: Fitness metrics, wellness habits
- **Career**: Professional development, achievements
- **Personal Development**: Learning completed, skills gained
- **Relationships**: Quality time, connections maintained

### 3. Project Evaluation (20 minutes)
- Review completed projects
- Assess active projects progress
- Archive old projects
- Plan new projects for next month

### 4. System Maintenance (15 minutes)
- Clean up Resources folder
- Update documentation
- Review and refine processes
- Check integration sync logs

### 5. Goal Review (15 minutes)
- Review short-term goal progress
- Update medium-term goals
- Adjust long-term goals if needed
- Set next month's focus areas

### 6. Planning & Reflection (10 minutes)
- Set 3 goals for next month
- Identify areas needing attention
- Celebrate wins
- Key learnings

## Monthly Review Checklist

- [ ] Complete financial review (budget vs actual)
- [ ] Update all Area files
- [ ] Archive completed projects
- [ ] Review quarterly goals (every 3 months)
- [ ] Clean up Resources and Archive
- [ ] Check integration status
- [ ] Set 3 goals for next month
- [ ] Reflect on month's progress

## Key Metrics to Track

### Financial
- Income vs budget
- Expenses by category
- Savings rate
- Net worth change

### Productivity
- Projects completed
- Tasks completed
- Review consistency
- System usage

### Personal Growth
- Books read
- Courses completed
- Skills practiced
- Habits maintained

## Tips for Effective Reviews

1. **Schedule it** - First of month, same time
2. **Block time** - Protect 1-2 hours
3. **Be thorough** - This is big picture time
4. **Use data** - Look at actual numbers
5. **Be honest** - Face reality
6. **Celebrate** - Acknowledge progress
7. **Adjust** - Make changes as needed

## Common Pitfalls

- **Skipping financial review** - Money management suffers
- **Not updating goals** - Goals become stale
- **No celebration** - Miss motivation boost
- **Too critical** - Discourages progress
- **No adjustments** - Repeat same mistakes
