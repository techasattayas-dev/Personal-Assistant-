# Literature Notes

## Purpose

Summaries and insights from external sources: books, articles, videos, podcasts, courses.

## How to Use

When consuming content:
1. **Capture source information** - Author, title, date, URL
2. **Extract key insights** - Main points and takeaways
3. **Add your thoughts** - How does this connect to what you know?
4. **Identify concepts** - Ideas worth developing into permanent notes

## Categories

Organize by source type or topic:
- Books/
- Articles/
- Videos/
- Courses/
- Research/
