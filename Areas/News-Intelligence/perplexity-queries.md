## Tips for Using Perplexity Effectively

### Best Practices
1. **Be Specific**: Include exact timeframes (past 24 hours, past week)
2. **Name Companies**: Mention specific competitors for better targeting
3. **Geographic Focus**: Specify Thailand, Myanmar, or regions clearly
4. **Industry Terms**: Use technical terms (feedmill, biosecurity, pelleting)
5. **Actionable Focus**: Ask for developments, changes, trends (not history)

### Time-Saving Tips
1. **Save Queries**: Bookmark frequently used queries in Perplexity
2. **Use Collections**: Organize queries by topic in Perplexity Collections
3. **Copy-Paste**: Keep this file open for quick copy-paste
4. **Batch Processing**: Run all morning queries together (10-12 minutes total)
5. **Follow-up**: Use "Ask follow-up" in Perplexity for deeper dives

### Quality Control
1. **Check Sources**: Verify Perplexity cites reputable sources
2. **Cross-Reference**: If critical, verify with additional search
3. **Date Check**: Ensure information is actually recent
4. **Bias Awareness**: Multiple sources reduce bias
5. **Local Context**: Verify international news applies to Thailand/Myanmar
