# Canton Fair 2026 — Visit Planning & Checklist

---

## Pre-Visit Preparation

### Registration & Access
- [ ] Register online at [cantonfair.org.cn](https://www.cantonfair.org.cn/en-US) (free)
- [ ] Download Canton Fair APP (iOS/Android) for catalog browsing
- [ ] Browse online catalog to pre-select target exhibitors
- [ ] Print/save buyer badge confirmation
- [ ] Prepare business cards (bilingual: English + Chinese recommended)

### Travel & Accommodation
- [ ] Book flights to Guangzhou Baiyun International Airport (CAN)
- [ ] Book hotel near Pazhou area (Haizhu District recommended)
- [ ] Arrange airport transfer or note metro route (Line 3 → Line 8 → Pazhou Station)
- [ ] Apply for China visa (if required)
- [ ] Arrange travel insurance
- [ ] Exchange currency (Chinese Yuan / RMB)
- [ ] Set up VPN for international internet access in China
- [ ] Download WeChat (essential for communication in China)
- [ ] Set up Alipay/WeChat Pay (mobile payments widely used)

### Team Preparation
- [ ] Assign team members to specific phases/sections
- [ ] Brief management team on target product categories
- [ ] Distribute phase research documents to all team members
- [ ] Set daily meeting points and communication plan
- [ ] Prepare product requirement specifications for suppliers
- [ ] Print supplier tracking sheets for each team member

| Time | Activity |
|------|----------|

## Supplier Evaluation Criteria

When evaluating potential suppliers or product ideas, use this scoring framework:

| Criteria | Weight | Score (1-5) | Notes |
|----------|--------|-------------|-------|
| Product Quality | 25% | | |
| Innovation Level | 20% | | |
| Price Competitiveness | 15% | | |
| MOQ Flexibility | 10% | | |
| Customization Capability | 10% | | |
| Certifications (ISO, CE, etc.) | 10% | | |
| Communication & English Level | 5% | | |
| Export Experience | 5% | | |
| **Total Weighted Score** | **100%** | | |

**Rating Guide:**
- 5 = Excellent — Immediately actionable
- 4 = Good — Worth follow-up discussion
- 3 = Average — Keep as alternative
- 2 = Below Average — Only if no better option
- 1 = Poor — Not suitable

### Short-term (Within 1 month)
- [ ] Present findings to broader management team
- [ ] Create product development recommendation report
- [ ] Request samples from top 10 suppliers
- [ ] Begin supplier due diligence for shortlisted partners
- [ ] Map product ideas to existing business strategy

- [ ] Negotiate terms with selected suppliers
- [ ] Initiate product development prototyping
- [ ] Plan follow-up visits or virtual meetings
- [ ] Register for Autumn 2026 session (140th) if needed
- [ ] Implement management process improvements identified
