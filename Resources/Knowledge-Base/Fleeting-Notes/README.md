# Fleeting Notes

## Purpose

Quick captures of ideas, thoughts, and insights as they occur. These are temporary by nature and should be processed within 7 days.

## How to Use

1. **Capture immediately** - Don't let ideas slip away
2. **Keep it brief** - Just enough to remember the thought later
3. **Don't worry about quality** - This is a brain dump
4. **Process regularly** - Review weekly and convert to permanent notes or discard

## Processing Workflow

Weekly (or more frequently):
- Review all fleeting notes
- Identify ideas worth developing
- Create permanent notes from valuable insights
- Delete or archive processed fleeting notes

## Template

```markdown
---
title: Quick idea description
created: 2026-03-11
type: fleeting
---

The idea in brief...
```
