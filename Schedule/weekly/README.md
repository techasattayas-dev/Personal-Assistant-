# Weekly Reviews

## Purpose

Weekly reviews are your comprehensive system maintenance time. This is when you:
- Process everything in your Inbox to zero
- Review all active projects
- Archive completed items
- Plan the upcoming week
- Reflect on progress

## Schedule

**Day**: Sunday
**Time**: 7:00 PM
**Duration**: 30-60 minutes

## File Naming

Files are named by year and week number: `YYYY-WXX.md`

Example: `2026-W11.md` (Week 11 of 2026)

## Weekly Review Structure

### 1. Inbox Processing (15 minutes)
- Empty quick-capture.md
- Empty inbox.md
- Empty to-process.md
- All items categorized or deleted

### 2. Project Review (15 minutes)
- Review each active project
- Update project status
- Add/remove tasks as needed
- Archive completed projects

### 3. Area Review (10 minutes)
- Check each Area for updates needed
- Finance: Review budget vs actual
- Health: Assess fitness goals
- Career: Note professional development
- Personal-Development: Review learning progress
- Relationships: Plan meaningful connections

### 4. Calendar Review (5 minutes)
- Review upcoming week's events
- Block focus time
- Identify potential conflicts

### 5. Weekly Planning (10 minutes)
- Set 3 goals for next week
- Identify key projects to focus on
- Plan time allocation
- Schedule important tasks

### 6. Reflection (5 minutes)
- What went well?
- What could improve?
- Key learnings
- Gratitude

## Weekly Review Checklist

- [ ] Process Inbox to zero
- [ ] Review all active projects
- [ ] Update Area files
- [ ] Archive completed items
- [ ] Review next week's calendar
- [ ] Set 3 goals for next week
- [ ] Reflect and write insights

## Tips for Effective Reviews

1. **Protect the time** - Make it non-negotiable
2. **Find a quiet space** - Minimize distractions
3. **Use a checklist** - Don't skip steps
4. **Batch similar tasks** - Review all projects together
5. **Be honest** - Face reality of progress (or lack thereof)
6. **Celebrate wins** - Acknowledge accomplishments

## Common Pitfalls

- **Skipping reviews** - Leads to system breakdown
- **Rushing through** - Quality matters
- **Not processing Inbox** - Defeats the purpose
- **Too perfectionist** - Good enough is fine
- **No reflection** - Miss opportunities for learning
